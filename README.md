#  DriveCar — Сервис проката автомобилей

## Описание проекта

DriveCar — это веб-приложение для аренды автомобилей. Пользователи могут просматривать каталог автомобилей, регистрироваться, авторизоваться и бронировать автомобили. Администратор может управлять автопарком через защищённый API.

---

```

│   ├── HTML/               # Страницы сайта

│   ├── CSS/                # Стили

    │   ├── carRoutes.js    # Маршруты автомобилей
```

- **Frontend:** HTML, CSS, JavaScript (Vanilla)
- **Backend:** Node.js, Express.js
- **База данных:** SQLite (sqlite3)

## Установка и запуск

### 1. Клонировать репозиторий
```bash

```

### 2. Установить зависимости
```bash
npm install
```

### 3. Запустить сервер
```bash
node app.js
```

### 4. Открыть в браузере
```
http://localhost:3000
```

## API маршруты

| Метод | Маршрут | Описание |
|-------|---------|----------|
| GET | /api/cars | Получить все автомобили |
| GET | /api/cars/:id | Получить автомобиль по ID |
| POST | /api/users/register | Регистрация пользователя |
| POST | /api/users/login | Вход пользователя |
| POST | /api/admin/login | Вход администратора |
| POST | /api/admin/cars | Добавить автомобиль (admin) |
| PUT | /api/admin/cars/:id | Обновить автомобиль (admin) |
| DELETE | /api/admin/cars/:id | Удалить автомобиль (admin) |
